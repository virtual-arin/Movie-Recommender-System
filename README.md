# 🎬 Movie Recommendation System

A content-based movie recommendation system that suggests similar movies based on user input.

## 🔧 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- OMDb API

## ⚙️ Features
- Recommend similar movies
- Cosine similarity-based recommendations
- Fetch movie posters using OMDb API
- Interactive Streamlit UI

## 🚀 How to Run

```bash
git clone https://github.com/virtual-arin/Movie-Recommendation-System-Project.git
cd Movie-Recommendation-System
pip install -r requirements.txt
streamlit run app.py

🔑 API Setup
Get API key from: http://www.omdbapi.com/apikey.aspx
Add in code:
            api_key = "YOUR_API_KEY"

📌 Project Files
app.py – Streamlit app
model.pkl – similarity model
movies.pkl – dataset
